# Awesome IINA

IINA is the modern media player for macOS.

## Contents

- [Plugins](#Plugins) 

## Plugins

- [iina/plugin-opensub](https://github.com/iina/plugin-opensub) for opensubtitles.com
- [iina/plugin-more-seeking](https://github.com/iina/plugin-more-seeking) Add more seeking options
- [iina/plugin-online-media](https://github.com/iina/plugin-online-media) 
- [5thDimensionalVader/recorder-iina](https://github.com/5thDimensionalVader/recorder-iina) clip a video using ffmpeg
- [bbeny123/iina-save-subs-with-delay](https://github.com/bbeny123/iina-save-subs-with-delay) Save subtitles with delay
- [bbeny123/iina-hdr-screenshot](https://github.com/bbeny123/iina-hdr-screenshot) Capture HDR Screenshot using FFmpeg
- [Robpol86/iina-file-name-search](https://github.com/Robpol86/iina-file-name-search) search the currently playing file's name on a website
- [bbeny123/iina-jump-to-frame](https://github.com/bbeny123/iina-jump-to-frame) Jump to Frame
- [mhajder/iina-jellyfin](https://github.com/mhajder/iina-jellyfin) for Jellyfin: automatic subtitles and media browser sidebar
- [wyattowalsh/iina-plugin-bookmarks](https://github.com/wyattowalsh/iina-plugin-bookmarks) media bookmark manager plugin for the iina video player
